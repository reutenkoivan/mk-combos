# US-001: Перший запуск

## Метадані

- ID: `US-001`
- Категорія: `Контексти входу / Перший запуск`
- Статус деталізації: `Описано`
- Пов'язані сценарії: `US-002`, `US-003`, `US-008`, `US-009`, `US-021`, `US-023`

## Архітектурний контекст

Root first launch працює через `apps/web`. Installed games приходять із `apps/web/src/game-business/installed-games/value.ts`.

First-launch setup зберігає default settings. Active game для route-prefixed deep link визначається URL, а не попередніми settings.

## Мета користувача

- Швидко зрозуміти, що застосунок є каталогом Mortal Kombat combos.
- Обрати default language, default game і notation display mode.
- Перейти до каталогу default game після підтвердження.

## Передумови

- Користувач відкриває root URL застосунку.
- У браузері немає збережених settings або first-launch completion marker.
- Застосунок має installed game business entry points для `MKXL` і `MK1`.

## Основний flow

1. Користувач відкриває root URL.
2. App Shell визначає, що це root first launch.
3. Користувач бачить required first-launch setup.
4. Setup пропонує вибрати language `EN/UA`, default game із `installedGames` і display mode `FGC/PlayStation/Xbox`.
5. Користувач підтверджує налаштування.
6. App зберігає settings або session-only equivalent.
7. App відкриває `/:gameId/catalog` для default game.
8. Далі користувач проходить active game catalog flow.

## Deep Link Bypass

Якщо fresh browser відкрив valid route-prefixed URL, наприклад `/mkxl/catalog` або `/mk1/combos/seeded/...`, `UI-PAGE-002` не показується.

У цьому flow App Shell:

- бере `gameId` із route prefix;
- перевіряє, що `gameId` є в `installedGames`;
- застосовує URL-derived active game;
- застосовує default display mode `FGC`;
- створює first-launch completion marker або session-only equivalent;
- відкриває target surface.

## Альтернативні flow

- Користувач приймає preselected defaults, але все одно має явно підтвердити setup.
- `localStorage` недоступний: app дозволяє session-only settings і пояснює обмеження.
- Контролер уже підключений: app може показати controller hint, але це не блокує setup.

## Out of scope

- Повноцінний onboarding wizard.
- Імпорт backup.
- Створення custom combo.
- Вибір game-specific catalog context у setup.

## Acceptance Criteria

- Root first-launch setup неможливо пропустити.
- Default game options беруться з `installedGames`.
- Після підтвердження app відкриває `/:gameId/catalog`.
- Valid route-prefixed deep link у fresh browser bypass-ить setup.
- Route prefix перемагає default game settings для active surface.
- Reload після підтвердження не показує setup повторно.

## Test Scenarios

- Fresh root URL показує required first-launch setup.
- Вибір `UA`, `MK1`, `Xbox` і confirmation відкриває `/mk1/catalog`.
- Reload після confirmation не показує setup.
- Fresh `/mkxl/catalog` не показує setup і відкриває MKXL catalog.
- Fresh `/mk1/catalog` не показує setup і відкриває MK1 catalog.
- Invalid unknown `/:gameId/...` не bypass-ить у робочу surface без recoverable error.

## Assumptions

- Browser locale `uk` або `uk-*` дає default language `UA`; усі інші locale дають `EN`.
- Default display mode - `FGC`.
- Default game може бути preselected, але не застосовується як completed first launch без confirmation.
